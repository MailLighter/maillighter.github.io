## MailLighter en bref

MailLighter est un plugin Outlook pense pour simplifier la gestion des emails et reduire la surcharge de messagerie au quotidien.

Ce depot contient le site public du projet:
- presentation produit,
- pages d'information,
- documentation d'usage.

Le site est developpe avec Astro pour offrir de bonnes performances, une base maintenable et une navigation fluide.

```bash

npm run dev
```

Le site est ensuite accessible sur:
http://localhost:4321
