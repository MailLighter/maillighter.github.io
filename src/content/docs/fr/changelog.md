## v1.0.0

Première version publique de MailLighter.

### Ajouté

- **Supprimer les images** : retire toutes les images intégrées du corps de l'e-mail, avec notification de l'espace libéré
- **Supprimer les pièces jointes** : supprime toutes les pièces jointes (hors images intégrées), avec notification de l'espace libéré
- **Conserver 2 réponses** : réduit la conversation aux 2 dernières réponses, avec notification du nombre de réponses détectées
- **Ne garder que la sélection** : remplace le contenu de l'e-mail par le texte sélectionné uniquement (peut contenir des images)
- **Nettoyage complet** : exécute toutes les opérations en une seule action avec un résumé détaillé (images, pièces jointes, réponses, total économisé)
- **Paramètres** : panneau de préférences accessible depuis le menu déroulant, avec message écologique (pied-de-page écologique personnalisable pour les e-mails envoyés) et suivi des économies (cumul des données supprimées sur toutes les sessions, stocké localement)
- Support multilingue : anglais, français, espagnol — les libellés du ruban et les notifications s'adaptent à la langue d'affichage d'Outlook
- Traitement 100 % local — aucun serveur externe, aucune donnée ne quitte votre appareil
- Compatible avec Outlook Desktop (Windows, macOS) et Outlook sur le Web
- Open-source sous licence MIT
